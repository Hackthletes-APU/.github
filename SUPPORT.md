# Support

For help with APU Hackthletes projects, please use the most relevant channel.

| Topic | Where to go |
| --- | --- |
| Bugs | Open a [bug report issue](https://github.com/Hackthletes-APU/.github/issues/new?template=bug_report.yml) in the relevant repository |
| Feature ideas | Open a [feature request issue](https://github.com/Hackthletes-APU/.github/issues/new?template=feature_request.yml) |
| Documentation | Open a [docs improvement issue](https://github.com/Hackthletes-APU/.github/issues/new?template=docs_improvement.yml) |
| Security concerns | Read [SECURITY.md](SECURITY.md) and report privately — do not open a public issue |
| Contribution questions | Read [CONTRIBUTING.md](CONTRIBUTING.md) first, then open a discussion or issue if you are still unsure |
| Hacker Hub listings | Use the `Add Hackathon` issue template in the Hacker Hub repository |
| Urgent or private matters | Contact maintainers at **apuhackthletes@gmail.com** |
