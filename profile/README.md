# APU Hackthletes

[![Hackthletes-APU](https://img.shields.io/badge/GitHub-Hackthletes--APU-181717?logo=github)](https://github.com/Hackthletes-APU)

APU Hackthletes is a student-led technology and hackathon community at Asia Pacific University, Malaysia. We build projects, organise technical sessions, support student builders, and help more students explore software engineering, AI, product development, and hackathons.

## What we do

- Run hackathon-related community initiatives
- Build open-source student projects
- Share technical learning resources
- Support beginners who want to join hackathons
- Connect students with builders, mentors, and opportunities

## Featured project

### Hacker Hub

Hacker Hub is a hackathon discovery and listing platform built by APU Hackthletes. It starts with Malaysian hackathons and aims to grow into a wider Asia and global hackathon discovery platform for students, builders, and organisers.

Browse all repositories: [github.com/Hackthletes-APU](https://github.com/Hackthletes-APU)
